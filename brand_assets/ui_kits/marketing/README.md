# Marketing Site — Daniel Rojas

A high-fidelity recreation of how `danielrojas.ai` plausibly looks given the
brand guidelines. **No production code was shared**, so these are
interpretations built directly from the brand primitives — palette, type,
button styles, voice, iconography rules. Treat as a starting point.

## Files
- `index.html` — full landing page (hero, proof row, modules, testimonial, CTA)
- `Nav.jsx` — sticky top navigation
- `Hero.jsx` — Silkscreen hero with Ember CTA
- `ProofRow.jsx` — Fortune-500 logo strip
- `ModuleCard.jsx` — course module card
- `Testimonial.jsx` — case-real testimonial block
- `CTASection.jsx` — closing call-to-action band
- `Footer.jsx` — three-column footer
