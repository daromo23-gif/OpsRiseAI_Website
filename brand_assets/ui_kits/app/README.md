# Aula — App UI Kit (interpretation)

A hypothetical learning + AI-workspace app called **Aula** ("classroom" in
Spanish) for Daniel Rojas's cohort program. **No production app code was
shared**, so this is an interpretation grounded in the brand primitives.

## Files
- `index.html` — Aula home (sidebar + top bar + agent chat + flows grid)
- `Sidebar.jsx` — left navigation
- `TopBar.jsx` — breadcrumb + search + new-flow CTA
- `FlowCard.jsx` — saved AI workflow card
- `ChatComposer.jsx` — agent chat composer
